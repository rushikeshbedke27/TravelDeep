This is the updated version the Travel Model:

To run this project follow below steps:

Create the Virtual Environment:
Step1: python -m venv tourismDeep_venv

Step2: Activate tourismDeep_venv virtual Environment(venv)
	tourismDeep_venv\Scritps\activate

Step3: Install all requirements.
	pip install -r requirements.txt

Step4: Run the Project:
	streamlit run app.py

Step5: i. After running the project please select the path for file "best_model.h5" and "final_df.csv". Please check double quote is not present in the path. ex: d:\project\best_model.h5

Step6: Then click on Initialize System to run the System.
